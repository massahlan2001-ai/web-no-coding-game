# 🎮 No Coding Game

An interactive web-based game where players click on moving targets to earn points. No coding skills required to play!

## 🎯 Game Features

- **Click-based gameplay** - Click on targets as fast as you can
- **Progressive difficulty** - Game gets harder with each level
- **Combo system** - Build combos for more points
- **Lives system** - 3 lives to survive
- **Sound effects** - Audio feedback for actions
- **Responsive design** - Works on desktop and mobile
- **Pause/Resume** - Play at your own pace

## 🎮 How to Play

1. Click the **Start Game** button
2. Click on the red circles (🎯) that appear in the game area
3. Earn points for each target clicked
4. Build combos for bonus points
5. Don't let targets disappear or you lose a life
6. Survive for 60 seconds to complete the level
7. Complete levels to increase the difficulty

## 📊 Scoring System

- **Base Points**: 10 points per target
- **Combo Bonus**: +5 points for each consecutive hit
- **Example**: 1st hit = 10pts, 2nd hit = 15pts, 3rd hit = 20pts

## ⚙️ Game Difficulty

- **Level 1**: 1-3 targets, spawning every 1 second
- **Level 2**: 2-4 targets, spawning every 0.9 seconds
- **Level 3+**: Increasing targets and spawn rate

The game dynamically adjusts difficulty based on your level:
- More targets appear simultaneously
- Targets spawn faster
- Targets disappear quicker

## 🎮 Controls

- **Mouse Click** - Click targets to score points
- **Space/Click** - Select buttons
- **ESC** - Pause/Resume during gameplay
- **Sound Button** - Toggle sound effects on/off
